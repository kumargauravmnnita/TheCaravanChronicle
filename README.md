The Caravan Chronicle

A municipal grievance tracking system for the Circus of Wonders.
Citizens can report issues like road damage, water leaks, and garbage problems.
Staff can manage and resolve them.
The Ringmaster (Admin) oversees everything.

Tech Stack

Backend: Node.js, Express.js, MongoDB Atlas
Frontend: React.js, Tailwind CSS
Image Upload: Cloudinary
Auth: JSON Web Tokens (JWT)
